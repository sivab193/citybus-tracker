An open-source, city-agnostic transit tracker. Point it at any GTFS feed and get a full-stack bot, API, and dashboard in a single docker compose up. Currently powering @cityb19_bot for Greater Lafayette, Indiana.

Features include live bus arrivals, smart subscription alerts with timezone-aware scheduling, favorite stops, full admin controls, an MCP server for AI assistants, and Swagger UI for interactive API exploration.
